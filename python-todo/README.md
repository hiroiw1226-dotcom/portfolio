# Todoアプリ（Python / ターミナル版）

## 概要
ターミナル（コマンドプロンプト）上で動くTodoアプリです。  
タスクの追加・一覧表示・完了状態の変更・削除ができます。

JavaScript 版の Todo アプリと同じ考え方を、Python でどう表現できるかを確認するために作りました。

## 機能
- `1` : タスクを追加する
- `2` : タスク一覧を表示する
- `3` : 指定したタスクを「完了」にする
- `4` : 指定したタスクを削除する
- `0` : アプリを終了する

タスクは Python の `list` に
